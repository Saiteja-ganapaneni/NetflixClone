# Netflix Clone (Demo)
This is a small React demo that mimics a Netflix-like UI (purely front-end, demo content).

## Run locally

1. Unzip the project (if zipped).
2. Install dependencies:
```
npm install
```
3. Start the development server:
```
npm start
```
Open http://localhost:3000 in your browser.

This project uses **react-scripts**. If you prefer Vite or another tool, you can adapt the code accordingly.
